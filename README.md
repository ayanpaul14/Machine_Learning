## 👨‍💻 About This Repo

This repository documents my **complete Machine Learning journey** — from Python fundamentals and NumPy basics all the way to building and deploying real ML models. Every concept is practiced hands-on in Jupyter Notebooks with clean code and output.

> 🎯 Goal: Master ML fundamentals and build production-ready models before placement season.

## 🛠️ Setup & Run Locally

```bash
# Clone the repo
git clone https://github.com/ayanpaul14/Machine_Learning.git
cd Machine_Learning

# Install dependencies
pip install numpy pandas matplotlib scikit-learn jupyter

# Launch Jupyter
jupyter notebook
```
