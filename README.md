## 👨‍💻 About This Repo

This repository documents my **complete Machine Learning journey** — from Python fundamentals and NumPy basics through data visualization, all the way to building real Scikit-Learn models and an end-to-end capstone project. Every concept is practiced hands-on in Jupyter Notebooks with clean code and output.

> 🎯 Goal: Master ML fundamentals and build production-ready models before placement season.

---

## 🛠️ Setup & Run Locally

```bash
# Clone the repo
git clone https://github.com/ayanpaul14/Machine_Learning.git
cd Machine_Learning

# Install dependencies
pip install numpy pandas matplotlib seaborn plotly cufflinks scikit-learn jupyter

# Launch Jupyter
jupyter notebook
```
